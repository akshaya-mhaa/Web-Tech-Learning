form tag is used to create a form like structure


label tag is used to display words or sentence inside the form


input tag is used to create a input field like a box where you enter your input

since it is a inline block the labels will get printed next to each other to avoid it br tag is used

if you want add border to the form use fieldset tag

attributes of input tag

input tag has attributes like type which in default is text which can be changed into number,email,password,textdate,etc another attribute is placeholder which displays sentence inside the input box



other attributes of input tag are id , if you give a name to id and use that id in label tag for attribute if you click on label then the border of the input tag will change


another attribute of input tag is name which is used to store the value enter in the specific input space


in checkbox we can select more than one box to add checkbox using label tag and add input attribute as type=checkbox to store the value of the checkbox labels we need to use value attribute and assign its value when it is selected


radio is a html tag unlike checkbox here only one option can be selected
