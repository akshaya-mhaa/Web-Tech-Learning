learning list in html

there are three types of list in html which are ordered list and unordered list , description list

Ordered list : Ordered list is crested using "ol" tag . Ordered list has a list of character before the attributes which are in sequencial order by default it has number from 1


the options for ordered list are numbers, alphabets(uppercase or lowercase),roman number(Uppercase or Lowercase)


you can change the default option by using "type="" " method and change type accordingly


you can also decide from where the numbers should start by using "start" method and assigning a start value


Unordered list: unordered list is created using "ul" tag . Undered list has symbols like circle , square,disc or you can choose nothing but default is has disc

you can change the symbols which are state before the attributes by using "type=" method and writing its type

  3. am
  4. dumb
